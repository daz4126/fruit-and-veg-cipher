🍏Fruit & Veg Cipher🥦

Mission:

Decrypt the secret enemy code word

Top secret intelligence:

The enemy are using pictures of fruit and veg to send secret messages.

The code is similar to a Polybius cipher and is made by writing a 5-letter keyword on the first row of a 5 x 5 grid then writing the rest of the alphabet after (I and J are considered the same letter).

Example: if the keyword was "RADIO", the cipher grid would look like this

🥦 🍆 🥔 🍄 🥕
🍏RADIO
🍌BCEFG
🍒HKLMN
🍉PQSTU
🍑VWXYZ

So the letter A would be 🍏🍆 and the letter Z would be 🍑🥕

Luckily some of our best analysts have been able to find some 5-letter words and the encryption to help, for example:

PRIDE = 🍉🥦 🍏🥦 🍏🍄 🍏🥔 🍌🥔

Be careful though - every check and clue generated will alert the enemy.
